# Contributing to JessRV1

Thank you for your interest in contributing to JessRV1.

## Getting Started

1. Fork this repository
2. Clone your fork

```bash
git clone https://github.com/your-username/JessRV1.git
```

3. Create a new branch

```bash
git checkout -b feature-name
```

## Development Guidelines

- Keep code clean and readable
- Follow existing project structure
- Test your changes before submitting
- Add comments where necessary

## Pull Request Process

1. Push your branch
2. Create a Pull Request
3. Describe the changes clearly
4. Wait for review

## Issues

Before starting work, comment on the issue and wait for assignment.

Thank you for contributing to JessRV1.
